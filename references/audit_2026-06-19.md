# Prompt Engagement Audit — 2026-06-19

Multi-agent comparison of current prompts against two successful YouTube gardening scripts (lavender, roses), with adversarial verification and a completeness critic.

## Stats

- **17 agents**: 5 lenses (retention, viewer-centered, language-craft, structural-rhythm, authority) × (analyze + audit + refute) + 1 completeness critic + 1 synthesizer
- **607K tokens**, **12.7 min** wall-clock
- **50 mechanics found**, 14 refuted, 36 surviving
- **113 proposed edits** across **13 prompt files**
- **11 missed patterns** + **10 vestigial rules** identified by completeness critic
- **7 documented conflicts**, all with embedded resolutions

## Headline finding

Our prompts encode the **intent** of the gardening voice (external villain, withheld reveals, "calmly certain" narrator) but consistently understate the **concrete moves** the samples use. Worse: a few existing rules — strict beat-and-word-count hook scaffolding, "no recap" conclusion, named-authority preference in listicle/research — **actively fight the voice** and need to be loosened or inverted.

Highest leverage edits concentrate in three files where ~30 surviving mechanics + 11 completeness patterns converge:
- `script_base.md` (27 edits)
- `script_modules/listicle.md` (16 edits)
- `channels/gardening.md` (16 edits)

## Top 3 actions (the synthesizer's own pick)

1. **`script_base.md`** — replace strict Hook Beat 1/2/3 scaffold with an unordered element checklist (8 elements: accusatory pattern interrupt → felt-pain tableau → external-villain absolution → withheld promise → numbered curiosity bookmark → triadic end-state promise → transition thread). Pair with conclusion rewrite into three named beats (compressed-recap drumroll → single-action prescriptive close → identity beat + villain callback) at ~120-180 words. **Five lenses converged here.**

2. **`script_modules/listicle.md`** — rewrite item-section beats around a three-beat arc (reframe → mechanism → hard rule). Beat 4 inverts default from "named-authority anchor optional" to "mechanism is the proof; named-expert anchor is rare exception." Beat 5 promotes reframing metaphor from optional decoration to conceptual spine. New beats 6c (hard-rule closer, <10 words) + 8 (aphorism button).

3. **`channels/gardening.md`** — sharpen narrator voice (no-hedging declaratives + every-paragraph second-person + one-rung-above humility) + add ~14 new bullets covering plant-as-character-with-strategic-intent, show-the-picture (banning abstract health words like "thriving"), vague-they villain framing, anticipation scaffolding, folk-origin authority, domain vocabulary with humanizing tags, insider-status flip, permission-to-do-less, hands-in-the-soil commands, two-beat sentences, muscular verbs, echo phrase, transformation promise, nicknames.

## The 7 conflicts and their resolutions

All conflicts have embedded resolutions in the ranked edits below.

1. **Conclusion "no recap" vs compressed drumroll** → distinguish padded prose recap (banned) from rhythmic drumroll (required). Resolved in script_base.md rank 6 + listicle.md rank 13.
2. **Listicle "no recap" same conflict** → ban recap *segment* but mandate drumroll inside `conclusion.narration` field. Resolved in listicle.md rank 13.
3. **Hook strict beat scaffold vs unordered checklist** → scrap the strict scaffold; checklist absorbs incremental Beat 1/2/3 edits from individual lenses. Resolved in script_base.md rank 5 + outline_base.md rank 1.
4. **Mechanism vs named-authority default** → invert: mechanism is required, named expert is rare exception. Resolved in listicle.md rank 7 + outline_base.md rank 2.
5. **Research producing fact+source vs script throwing it away** → reframe research output as mechanism inventory + specifics + taxonomy + folksy quantifiers; source attribution is for verification only. Resolved in research.md rank 2.
6. **Hook archetypes mutual exclusivity vs samples blending them** → allow primary + secondary blending. Resolved in hook_archetypes.json rank 1 (architectural).
7. **Outline four-element ordered hook scheme vs unordered checklist** → replace with unordered checklist including end-state, numbered cliffhanger, felt-pain tableau. Resolved in outline_base.md rank 1.

## Surprising findings worth flagging

1. **"Earn every sentence" + 3-beat hook + word counts FIGHT the voice.** Both samples violate every word-count rule. The 75-word hook can't fit accusatory + absolution + felt-pain + withheld + numbered bookmark + end-state + triadic indictment.
2. **Named-authority preference is actively harmful for this channel.** Both samples use ZERO researcher names, ZERO study citations, ZERO dates/institutions. Authority comes from mechanism, folk-origin, domain vocabulary, calibrated specifics.
3. **"No recap" rule is wrong in detail.** Both samples DO recap — as compressed rhythmic drumrolls (one beat per item, parallel imperatives), not padded prose. Need to distinguish.
4. **Hook archetypes shouldn't be mutually exclusive.** Strongest hooks blend 2-3 archetypes.
5. **Anticipation scaffolding is a per-segment HABIT, not a one-off.** "Here's the part nobody mentions" should appear at least once per segment.
6. **Permission to do LESS is structurally significant.** Both samples' rescue is "stop doing X" not "add another step."
7. **Reframing metaphor as section SPINE, not decoration.** "Lavender is a tough Mediterranean weed wearing a pretty dress" does the conceptual work of an entire section.
